Each Markdown report (e.g., `CREATIVE_SHOWCASE_001.md`) includes a link to its HTML story

## What You'll See

### Story Header
- Pipeline name and description
- Execution timestamp
- Overall status

### Step Cards (One Per Pipeline Step)
Each card shows:
- **Step Name** (e.g., "ingest_csv", "transform_1")
- **Layer** (Bronze/Silver/Gold)
- **Execution Time**
- **Before Snapshot**: Input data schema + sample rows
- **After Snapshot**: Output data schema + sample rows
- **Schema Diff**: What changed (columns added/removed/type changes)
- **Row Delta**: How many rows were added/removed

### Interactive Features
- **Collapsible sections**: Click headers to expand/collapse
- **Sample data tables**: See actual data flowing through
- **Color coding**: Green for success, red for failures
- **Responsive design**: Works on desktop and mobile

## Technical Details

### Story Generator
Stories are generated by `Tracker.export_to_story()`:
- Captures snapshots before/after each step
- Computes schema diffs automatically
- Renders interactive HTML with embedded CSS
- Self-contained (no external dependencies)

### File Size
- Average story size: ~7-15 KB
- Includes embedded CSS (no external files needed)
- Sample data limited to first 5 rows for brevity

## Troubleshooting

**Q: Story HTML won't open?**  
A: Make sure you're opening it in a web browser (Chrome, Firefox, Edge)

**Q: No data shown in snapshots?**  
A: Current implementation simulates execution. Real data would show actual values.

**Q: Want to customize the story?**  
A: Edit `odibi_core/story/story_generator.py` to change templates
