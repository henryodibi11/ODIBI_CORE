## ğŸ› Troubleshooting

### "streamlit: command not found"
```powershell
pip install streamlit
# Or use:
python -m streamlit run learnodibi_ui.py
```

### "pyperclip not installed"
```powershell
pip install pyperclip
```

### "Still see pass statements"
```powershell
python fix_pass_statements.py
```

### "Light mode hard to read"
- Make sure you restarted the app after latest changes
- Clear browser cache (Ctrl+Shift+R)
- Latest fix should show white dropdowns with dark text

### "No lessons showing"
- Verify you're in `D:\projects\odibi_core` directory
- Check `docs/walkthroughs/` folder exists
- Should have 11 `DEVELOPER_WALKTHROUGH_*.md` files
